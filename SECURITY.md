# Security

Dot is a local desktop app. It does not send analytics and does not require an account.

If you find a security issue, please open a private security advisory on GitHub or contact the maintainer privately before posting public details.

## Privacy Notes

- Settings are stored locally in Electron's user data folder.
- Keyboard watch uses Electron global shortcuts and can be disabled.
- Dot should not collect keystroke contents or send user activity to a server.
