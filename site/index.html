Open-source desktop companion

Dot

A tiny pixel cat that lives on your Mac, watches your cursor, reminds you to stretch, and keeps your focus sessions company.

Download for macOS Apple Silicon View source

Unsigned open-source build for Apple Silicon Macs. macOS may block first launch. Windows support is planned.

Focus 24:13

Clean on your desktop

No buttons stuck to the cat. Dot is controlled from the menu bar, right-click menu, or settings window.

Local and tiny

Settings stay on your machine, and the pet is drawn in canvas with no external art assets.

Useful little nudges

Stretch reminders, Pomodoro focus, typing reactions, cursor tracking, full-screen hiding, color tweaks, and open-at-login.

Install

  1. Click `Download for macOS Apple Silicon` to download the `.dmg` directly.
  2. Open the `.dmg` and drag `Dot.app` to Applications.
  3. Launch Dot, then click the menu bar icon to open settings.

If macOS says Dot is damaged

Dot is an unsigned open-source build. If macOS blocks it after download, run this command after moving Dot to Applications:

xattr -dr com.apple.quarantine /Applications/Dot.app
open /Applications/Dot.app

Verify the download

SHA256 for `Dot-0.1.0-arm64.dmg`:

f98af460451119f23b61b467c3476230b170e3c323f82647354caa50dad2cbda

Build from source

If you do not want to run an unsigned download, build Dot locally from the source code:

git clone https://github.com/Anshumaan657/Dot.git
cd Dot
npm install
npm start
